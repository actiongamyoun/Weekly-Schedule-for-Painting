① 접속

② 검사 등록
1. 좌측에서 우리 회사명 클릭 선택
2. 캘린더 날짜 셀 클릭 또는 ＋ 검사 등록 버튼 클릭
3. 프로젝트·호선·검사 아이템·검사종류·날짜 입력 후 등록
4. 최초 1회 회사명 확인 후 같은 세션에서는 자동 적용
③ 수정 / 삭제
캘린더의 검사 카드 클릭 → 수정 또는 삭제
④ 주간 이동
날짜 옆 ‹ › 버튼으로 이전/다음 주 이동
